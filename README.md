# Calculadora Python

Calculadora gráfica sencilla creada con **Python** y la librería **Tkinter**.

### Características
* **Diseño Dark Mode:** Interfaz moderna en tonos oscuros.
* **Funciones:** Suma, resta, multiplicación y división.
* **Extras:** Botón para borrar último dígito (`←`) y botón para limpiar pantalla (`c`).

### Requisitos
No necesitas instalar nada extra. Solo tener **Python** instalado en tu PC, ya que usa librerías nativas.

### Cómo ejecutar
1. Descarga el archivo `.py`.
2. Abre la terminal en la carpeta del archivo.
3. Escribe: `python calculadora.py`
